Datavisualisation – Subventions par Organisme (Metabase)

Projet Data Viz — Metabase + PostgreSQL

  • Catégorie : Datavisualisation / Analytics
  • Stack : PostgreSQL Metabase Debian 12 DBeaver Git
  • Repo : github.com/avisse/metabase-digdash-databisualisation

Mise en place d’une chaîne locale de datavisualisation : création d’une base PostgreSQL, injection de données (500+ lignes), connexion Metabase et construction d’un dashboard métier pour suivre les subventions par organisme, par mois et par type de finalité.

Indicateurs clés
  • Montant total des subventions par organisme
  • Évolution mensuelle (2020–2024)
  • Nombre de subventions par organisme
  • Montant par finalité (objet)
1) Modèle de données & préparation ▼
  • Installation PostgreSQL (VM Debian 12) et création de la base subventions_db
  • Table subventions : id, organisme, objet, montant, date_versement
  • Génération et insertion de 500+ lignes réalistes (2020–2024)
  • Vérifications via DBeaver (typages, contraintes, index de base)
-- Total des subventions par organisme
SELECT organisme, SUM(montant) AS total
FROM subventions
GROUP BY organisme
ORDER BY total DESC;

-- Subventions mensuelles pour un organisme
SELECT date_trunc('month', date_versement) AS mois, SUM(montant) AS total
FROM subventions
WHERE organisme = 'Association Sportive'
GROUP BY mois
ORDER BY mois;
2) Configuration Metabase & dashboard ▼
  • Lancement de Metabase (localhost:3000) et connexion à PostgreSQL
  • Typage des champs (Date, Numérique) et synchronisation des métadonnées
  • Création de vues/collections Metabase + cartes, barres, séries temporelles
  • Dashboard : organismes, par mois, par finalité, nombre de dossiers
